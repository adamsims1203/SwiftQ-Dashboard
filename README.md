## SwiftQ-Dashboard

### What is it?
SwiftQ-Dashboard is a standalone Vapor app for monitoring Tasks from SwiftQ.  The Dashboard is built using Vapor 3.

### Features
* Node monitoring with failure events
* Task statistics
* Scheduled task insights
* Failed task logs
* Redis Statistics
	* Connected clients
	* Blocked clients
	* Memory usage
	* uptime
